## 计分说明

### 1. 大五维度计分

```
O 维度 = 题目 1-12 得分总和（注意反向题）
C 维度 = 题目 13-24 得分总和
E 维度 = 题目 25-36 得分总和
A 维度 = 题目 37-48 得分总和
N 维度 = 题目 49-60 得分总和
```

### 2. MBTI 维度计分

```
E-I 维度 = E 维度原始分 + 中文补充 E-I 题目分
N-S 维度 = O 维度原始分 + 中文补充 N-S 题目分
T-F 维度 = (6 - A 维度原始分) + 中文补充 T-F 题目分（反向）
J-P 维度 = C 维度原始分 + 中文补充 J-P 题目分
```

### 3. 类型判定

- 各维度得分 ≥ 中值 → 正向类型（E/N/T/J）
- 各维度得分 < 中值 → 反向类型（I/S/F/P）
