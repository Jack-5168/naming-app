## 技术指标

### IRT 模型
- **模型**: 2PL（双参数逻辑模型）
- **公式**: P(θ) = 1 / (1 + e^(-a(θ - b)))
- **参数范围**: a ∈ [0.5, 2.5], b ∈ [-2, +2]

### 质量标准
| 指标 | 目标值 | 验收标准 |
|------|--------|----------|
| 重测信度 | > 0.80 | > 0.75 |
| Cronbach's α | > 0.85 | > 0.80 |
| 题目区分度 | > 1.0 | > 0.8 |
| CFI | > 0.90 | > 0.85 |
| TLI | > 0.90 | > 0.85 |
| RMSEA | < 0.08 | < 0.10 |

### 预测试样本
- **目标**: N ≥ 500
- **重测子样本**: N ≥ 100（间隔 2-4 周）
- **预计剔除率**: 10-17%
