### V28.0.2 - 套餐系统版

**时间**: 2026-04-18 12:50  
**类型**: 🆕 新功能  
**状态**: ✅ 已完成

#### 变更概述
根据会员等级决定名字生成数量，实现商业化分层

#### 核心变更
| 项目 | 变更前 | 变更后 |
|------|--------|--------|
| 名字数量 | 固定 10 个 | 按套餐：3/10/30 个 |
| 限流策略 | 统一每日 3 次 | 仅免费用户限流 |
| Prompt | 固定 10 个 | 动态数量 |

#### 代码变更
**文件**: `index.html`

**新增函数**:
```javascript

    TIER_LIMITS: {
        'free': 3,
        'premium': 10,
        'deluxe': 30
    }
};

// 2. 当前套餐状态
let currentUserTier = localStorage.getItem('userTier') || 'free';

// 3. 获取套餐限制
function getNameLimitByTier() {
    return AI_CONFIG.TIER_LIMITS[currentUserTier] || AI_CONFIG.TIER_LIMITS.free;
}

// 4. 动态 Prompt
function buildNamingPrompt(formData) {
    const nameCount = getNameLimitByTier();
    return `请为以下宝宝起${nameCount}个名字：...`;
}

// 5. 套餐选择
function selectPricing(tier) {
    currentUserTier = tier;
    localStorage.setItem('userTier', tier);
    showUserFriendlyError(`已选择 ${tierNames[tier]}`);
}

// 6. 套餐显示

    // 在表单上方显示套餐徽章

// 付费用户不限次数
```
