```bash

git add .
git commit -m "🎉 Web 站 V30.0.0 完整版 - 8 个核心页面"
git push origin master
```

### 2. 配置 GitHub Pages
- 进入仓库 Settings → Pages
- Source 选择 `main` branch
- Folder 选择 `/web-v2`
- Save 保存

### 3. 访问地址
- **新 Web 站**: https://jack-5168.github.io/naming-app/web-v2/

### 4. 本地测试

# 访问 http://localhost:8080
