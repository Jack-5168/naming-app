# 人格测试预测试方案

## 研究概述

**研究目的**: 对 IPIP-NEO-60 + 中文补充题库进行 IRT 参数标定和信效度检验  
**研究设计**: 横断面调查 + 重测信度子样本  
**预计周期**: 6 周（Week 3-8）

---

### 样本量目标

| 样本类型 | 目标数量 | 用途 |
|----------|----------|------|
| 主样本 | N ≥ 500 | IRT 参数标定、信效度检验 |
| 重测子样本 | N ≥ 100 | 重测信度（间隔 2-4 周） |
| 预留损耗 | 20% | 应对无效问卷、失访 |

- **优势**: 样本同质性好，易于追踪重测
- **激励**: 学分/综测加分 + ¥10 红包 + 人格报告

#### 3. 朋友圈/社交网络（目标 N=150）
- 研究者朋友圈转发

- 微博/小红书话题
- **优势**: 样本多样性好
- **激励**: ¥10 红包 + 人格报告

### 纳入标准

✅ **必须满足**:
- 年龄 16-65 岁
- 中文阅读理解能力正常

### 重测流程（2-4 周后）

1. **联系方式收集**: 基线测试结束时收集微信/邮箱
2. **重测邀请**: 2 周后发送重测链接（短信/微信）
3. **重测测试**: 仅测试 IPIP-NEO-60（60 题，5 分钟）
4. **激励发放**: 完成重测后发放额外¥5 红包
5. **失访追踪**: 3 天后未回复则发送提醒，7 天后视为失访

## 五、质量控制

### 无效问卷剔除标准

符合以下**任一**条件即剔除：

1. **答题时间异常**
   - 总时间 < 5 分钟（过快）
   - 总时间 > 60 分钟（过长，可能中断）
   - 平均每题 < 1.5 秒

2. **注意力检测失败**
   - 3 题注意力检测题错误 ≥ 2 题

3. **反应定势**
   - 连续 20 题选择相同选项
   - 方差 < 0.5（几乎无变异）

4. **社会赞许性过高**
   - 测谎题总分 ≥ 11 分（满分 12）

5. **一致性检验失败**
   - 与 IPIP 对应题相关系数 < 0.3

6. **缺失值过多**
   - 缺失题数 > 10 题（5.6%）

### 数据清理流程

```python
def clean_data(raw_df):
    # 1. 剔除时间异常
    df = raw_df[
        (raw_df['duration'] >= 300) & 
        (raw_df['duration'] <= 3600)
    ]
    
    # 2. 剔除注意力检测失败
    df = df[df['attention_correct'] >= 2]
    
    # 3. 剔除社会赞许性过高
    df = df[df['social_desirability'] < 11]
    
    # 4. 处理缺失值
    # 缺失<5 题：均值插补
    # 缺失≥5 题：剔除
    df = handle_missing(df, threshold=5)
    
    # 5. 反向题转换
    df = reverse_score(df, reverse_items)
    
    return df
```
