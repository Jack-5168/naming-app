## 一、研究目的

本研究旨在对编制的 300 道 MBTI 测试题目进行预测试和 IRT 参数标定，为后续 CAT（计算机自适应测试）系统提供高质量题库。

### 具体目标

1. 验证题目的信度和效度
2. 估计题目的 IRT 参数（区分度 a、难度 b）
3. 筛选出高质量题目（区分度>0.8，难度在 -2 到 +2 之间）
4. 建立常模数据
5. 验证四因子结构模型

## 三、测试流程

### 3.1 整体流程

```
知情同意 → 人口学信息 → 正式测试 → 注意力检测 → 结束反馈
                ↓
          重测子样本（2-4 周后）
```

### 3.2 详细步骤

#### 步骤 1：知情同意（必填，约 2 分钟）

**内容**:
- 研究目的说明
- 测试时长说明（约 20-25 分钟）
- 数据保密承诺
- 自愿参与声明
- 联系方式

**知情同意书要点**:
```
1. 本研究旨在开发人格测评工具
2. 测试约需 20-25 分钟完成
3. 所有数据将匿名处理，仅用于研究
4. 您可随时退出测试
5. 完成测试后可获得人格报告和红包奖励
6. 如有疑问请联系：research@persona-lab.org
```

#### 步骤 2：人口学信息（约 3 分钟）

**必填项目**:
- 年龄（数字）
- 性别（男/女/其他/不愿透露）
- 教育程度（初中/高中/专科/本科/硕士/博士）
- 职业（学生/在职/自由职业/其他）
- 所在地区（省份）

**选填项目**:
- 月收入范围
- 婚姻状况
- 既往心理测评经历

#### 步骤 3：正式测试（约 20-25 分钟）

**题目数量**: 300 题（含 15 题测谎题）

**呈现方式**:
- 随机呈现（测谎题固定位置）
- 一次一题
- 不可返回修改

**选项**:
- A. 非常符合
- B. 比较符合
- C. 不太符合
- D. 非常不符合

**时间控制**:
- 最短时间：5 分钟（少于 5 分钟数据作废）
- 最长时间：60 分钟（超时自动提交）
- 平均时间：20-25 分钟

#### 步骤 4：注意力检测（嵌入在正式测试中，3 题）

**检测题示例**:
1. "本题请选择'非常符合'"（检测是否认真阅读）
2. "本题请选择'不太符合'"（检测是否认真阅读）
3. "本题请选择 A 选项"（检测是否认真阅读）

**处理规则**:
- 答错 0 题：通过
- 答错 1 题：通过（可能为误操作）
- 答错≥2 题：数据作废

#### 步骤 5：结束反馈（约 2 分钟）

**内容**:
- 感谢参与
- 预计报告发放时间
- 重测邀请（随机抽取 100 人）
- 联系方式

### 3.3 重测流程（2-4 周后）

**重测子样本**: N≥100

**流程**:
1. 通过短信/邮件邀请
2. 使用相同题库（题目顺序随机重排）
3. 测试时长：20-25 分钟
4. 激励：¥10 红包

**目的**:

## 四、质量控制

### 4.1 数据剔除标准

| 剔除标准 | 具体规则 | 预期剔除率 |
|----------|----------|------------|
| 答题时间过短 | <5 分钟 | 3-5% |
| 答题时间过长 | >60 分钟 | 1-2% |
| 注意力检测错误 | ≥2 题错误 | 2-3% |
| 社会赞许性过高 | 测谎题得分>80 | 2-3% |
| 作答模式异常 | 连续 20 题相同选项 | 1-2% |
| 缺失值过多 | 缺失>10% | 1-2% |
| **合计** | | **10-17%** |

### 4.2 具体检测方法

#### 4.2.1 答题时间检测

```python
def check_response_time(df):
    """
    检测答题时间
    有效范围：300 秒 - 3600 秒（5-60 分钟）
    """
    min_time = 300  # 5 分钟
    max_time = 3600  # 60 分钟
    
    valid = (df['total_time'] >= min_time) & (df['total_time'] <= max_time)
    return valid
```

#### 4.2.2 注意力检测

```python
def check_attention(df):
    """
    检测注意力检测题
    允许错误 0-1 题，≥2 题错误则剔除
    """
    attention_items = [100, 200, 250]  # 注意力检测题题号
    correct_answers = {'100': 'A', '200': 'C', '250': 'A'}
    
    error_count = 0
    for item in attention_items:
        if df[f'item_{item}'] != correct_answers[str(item)]:
            error_count += 1
    
    return error_count <= 1
```

#### 4.2.3 社会赞许性检测

```python
def check_social_desirability(df):
    """
    检测社会赞许性
    测谎题 301-305，得分>80 则剔除
    """
    lie_items = [301, 302, 303, 304, 305]
    lie_score = sum([df[f'item_{item}'] for item in lie_items])
    
    # 测谎题满分 500，>80% 表示过度伪装
    return lie_score <= 400
```

#### 4.2.4 作答模式检测

```python
def check_response_pattern(df):
    """
    检测异常作答模式
    连续 20 题相同选项则剔除
    """
    items = [f'item_{i}' for i in range(1, 301)]
    responses = [df[item] for item in items]
    
    max_consecutive = 1
    current_consecutive = 1
    
    for i in range(1, len(responses)):
        if responses[i] == responses[i-1]:
            current_consecutive += 1
            max_consecutive = max(max_consecutive, current_consecutive)
        else:
            current_consecutive = 1
    
    return max_consecutive < 20
```

#### 4.2.5 缺失值检测

```python
def check_missing_values(df):
    """
    检测缺失值
    缺失>10% 则剔除
    """
    items = [f'item_{i}' for i in range(1, 301)]
    missing_count = sum([df[item].isna() for item in items])
    
    return missing_count <= 30  # 300 题的 10%
```

### 4.3 数据清洗流程

```
原始数据 (N=500)
    ↓
剔除答题时间异常 (约 25 人)
    ↓
剔除注意力检测失败 (约 15 人)
    ↓
剔除社会赞许性过高 (约 15 人)
    ↓
剔除作答模式异常 (约 10 人)
    ↓
剔除缺失值过多 (约 10 人)
    ↓
有效数据 (N≈425)
```

## 五、数据分析计划

### 5.1 描述性统计

- 样本人口学特征分布
- 各维度得分分布
- 题目难度和区分度初步估计

### 5.2 信度分析

1. **内部一致性信度**
   - Cronbach's α系数
   - 分半信度
   - 各维度α系数

2. **重测信度**
   - 重测相关系数
   - 重测一致性百分比

### 5.3 效度分析

1. **结构效度**
   - 探索性因子分析（EFA）
   - 验证性因子分析（CFA）
   - 四因子模型拟合指数

2. **内容效度**
   - 专家评定
   - 题目 - 维度匹配度

### 5.4 IRT 参数标定

1. **模型选择**: 2PL（双参数逻辑模型）
2. **参数估计方法**: MMLE（边缘最大似然估计）
3. **参数范围**:
   - 区分度 a: 0.5-2.5
   - 难度 b: -2 到 +2

### 5.5 题目筛选标准

| 标准 | 阈值 | 处理 |
|------|------|------|
| 区分度 a | <0.8 | 删除 |
| 难度 b | >2 或 <-2 | 删除 |
| 猜测参数 c | >0.3 | 删除 |
| 题目 - 总分相关 | <0.3 | 删除 |
| 选项分布 | 某选项<5% | 修改或删除 |
