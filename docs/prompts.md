## 基础报告模板 (BASIC_REPORT_TEMPLATE)

**适用场景**: 快速人格解读
**字数要求**: 800-1200 字
**Token 预算**: 1500

```

- 各维度得分：E={{eScore}}%, N={{nScore}}%, T={{tScore}}%, J={{jScore}}%

1. 用温暖、专业的语气撰写
2. 避免绝对化表述（不使用"保证"、"一定"、"绝对"等词）

   - 核心特质解读（约 200 字）
   - 优势与潜力（约 200 字）
   - 成长建议（约 200 字）
   - 适合的职业方向（约 150 字）

4. 在报告末尾添加免责声明："本报告基于心理学理论生成，仅供参考，不构成专业心理评估。"
5. 总字数控制在 800-1200 字之间
```

## 模板使用说明

### 变量替换

模板支持以下变量：

| 变量 | 说明 | 示例 |
|------|------|------|
| `{{mbtiType}}` | MBTI 人格类型 | INTJ |
| `{{eScore}}` | 外向维度得分 | 45 |
| `{{nScore}}` | 直觉维度得分 | 72 |
| `{{tScore}}` | 思考维度得分 | 68 |
| `{{jScore}}` | 判断维度得分 | 81 |
| `{{testDate}}` | 测试日期 | 2026-04-21 |
| `{{#lifeEvents}}...{{/lifeEvents}}` | 生活事件列表 | 见下方 |

### 生活事件格式

```
{{#lifeEvents}}
- {{date}}: {{title}} ({{type}}) - {{description}}
{{/lifeEvents}}
```

完整事件对象包含：
- `date`: 事件日期
- `title`: 事件标题
- `type`: 事件类型（career/relationship/health 等）
- `description`: 事件描述
- `expectedImpact`: 预期影响（positive/negative/neutral）
- `actualImpactScore`: 实际影响评分（-100 到 100）
- `relatedDimension`: 相关人格维度（E/N/T/J）

## 质量检查清单

生成报告后，请检查：

- [ ] 字数符合要求（基础 800-1200，专业 2000-3000，大师 4000-5000）
- [ ] 不包含禁止词（"保证"、"一定"、"绝对"、"肯定"、"必然"、"无疑"）
- [ ] 包含免责声明
- [ ] 所有章节标题齐全
- [ ] 语气符合报告类型（温暖/专业/大师级）
- [ ] 生活事件与人格关联标注为"探索性分析"
