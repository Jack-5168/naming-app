## 功能详情

### 分享海报
**内容结构：**
```
文昌赐名 · 专业起名
    既明
  ⭐ 98 分
《诗经·大雅·烝民》
已然明智，天赋异禀
   [二维码]
  文昌赐名
```

**技术栈：**
- html2canvas v1.4.1
- 离屏渲染
- 2x 高清缩放

**使用场景：**
- 微信朋友圈分享
- 家庭群讨论
- 社交媒体传播

---

### PDF 报告
**报告结构：**
```
【封面】标题
【名字】姓名 + 评分
【典籍】出处 + 原文
【寓意】解读
【音韵】拼音/声调/平仄/音律
【八字】四柱 + 五行
【页脚】品牌信息
```

**技术栈：**
- jsPDF v2.5.1
- A4 格式 (210x297mm)
- 手动排版

**待优化：**
- ⚠️ 中文字体支持
- 解决方案：加载 Noto Sans SC

---

### 名字对比
**对比维度（10 个）：**
1. 名字
2. 评分
3. 典籍出处
4. 寓意
5. 拼音
6. 声调
7. 平仄
8. 音律
9. 五行
10. 八字分析

**交互流程：**
1. 勾选复选框（至少 2 个）
2. 点击"对比选中名字"
3. 查看对比表格
4. 关闭弹窗做决策

**使用场景：**
- 多个名字难以抉择
- 家庭讨论对比
- 专业分析参考

## 已知问题

### PDF 中文字体 ⚠️
**问题**: jsPDF 默认字体不支持中文

**影响**: 中文字符可能显示为乱码

**解决方案**:
1. 加载自定义字体文件（推荐）
2. 使用 pdfMake 库替代
3. 后端生成 PDF 服务

**临时方案**: 优先使用海报功能（PNG 无此问题）
